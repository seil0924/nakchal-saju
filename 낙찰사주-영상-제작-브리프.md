# 낙찰사주 — 히어로 영상 제작 브리프

목표: **"와, 이 사이트 뭐지?"** 첫 3초에 사람을 홀리는 프리미엄 영상.
전 영상 공통 톤 — 먹빛(ink black) 위에 금박(gold leaf)·인주(vermillion) 액센트. 느리고, 매크로(macro)하고, 얕은 심도(shallow DoF). 박물관급 정물 + 명리(命理)의 신비.

| 해상도 | **3840×2160(4K) 권장**, 최소 1920×1080 |

## 3. 납품 전 압축 (용량 맞추기)
4K 원본을 웹용으로 줄이는 ffmpeg 명령 (개당 ~2MB, 무음, faststart):

```bash
ffmpeg -i hero-1_master.mp4 -an -vf "scale=1920:-2" \
  -c:v libx264 -pix_fmt yuv420p -crf 26 -preset slow -movflags +faststart hero-1.mp4
```
포스터 추출:
```bash
ffmpeg -i hero-1.mp4 -vf "select=eq(n\,0),scale=1280:-2" -q:v 3 hero-1-poster.jpg
```
